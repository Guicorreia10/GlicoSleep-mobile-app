# 🩺 GlicoSleep

Aplicação mobile para monitorização de níveis de diabetes, com integração de dados de sono e análise inteligente de padrões.

🏆 Projeto final de licenciatura classificado com **18/20 valores**

---

## 🚀 Funcionalidades

* 📊 Registo de níveis de glicemia
* 😴 Monitorização de sono (manual e via wearables)
* 🍽️ Registo alimentar com recurso a visão computacional
* 📈 Histórico de medições e evolução
* 🤖 Geração de insights com base em IA
* 🔵 Conectividade Bluetooth com dispositivos wearable
* 📱 Interface simples e intuitiva

---

## 🛠️ Tecnologias

**Mobile:**
React Native, Expo

**Backend & Dados:**
Node.js, Supabase

**Outras:**
Visão Computacional, Inteligência Artificial

## 🧠 Engenharia de Requisitos

Este projeto foi desenvolvido com base em práticas de Engenharia de Requisitos, incluindo:

* 📌 Identificação de stakeholders
* 📋 Definição de requisitos funcionais e não funcionais
* 📐 Modelação do sistema (UML e casos de uso)

## 💡 Objetivo

Apoiar utilizadores na monitorização da diabetes, promovendo uma melhor compreensão da relação entre glicemia, alimentação e qualidade do sono.
